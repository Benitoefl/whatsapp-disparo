# WhatsApp Disparo Automático

Automação para envio de mensagens pelo WhatsApp Web com base em uma planilha Excel.

## Estrutura
whatsapp-disparo/
├── main.py
├── requirements.txt
├── .gitignore
└── data/
└── Disparo.xlsx

markdown
Copiar código

## Como usar
1. Instale dependências: `pip install -r requirements.txt`
2. Coloque sua planilha em `data/Disparo.xlsx`
3. Abra o WhatsApp Web no navegador padrão e faça login.
4. Execute: `python main.py`

⚠️ Use apenas para comunicações legais e com consentimento.
